# DenpaModem

**DenpaModem** is an open-source project focused on developing a programmable baseband modem for amateur radio applications.

The goal is to provide a flexible platform capable of generating digital modulation waveforms directly from embedded hardware, without relying on traditional sound cards or dedicated RF modem ICs.

> **Project status:** Early development. Nothing is implemented yet.
